Entropy=L×log(N) base 2

where:

L = password length
N = size of the character pool

Example

Password: P@ssw0rd123

Length = 11

Charset =

26 lowercase
26 uppercase
10 digits
32 symbols

N = 94

Entropy = 11 × log₂(94)

≈ 72 bits

The problem?

It says

AAAAAAAAAAAA

has exactly the same entropy as

T#8L$2q@Xp!K

which is obviously false.
